How it works

Calculators: Input geometry and loads. Formulas follow standard mechanics (BM = WL/4 for point load). Concrete design uses simplified Limit State checks.

Sketchpad: Click and drag to draw lines. Use the scale input to calibrate (e.g., if 100 pixels = 5m, set scale to 0.05).

Takeoff: Add rows manually. Results update automatically. Export PDF creates a formatted report.

Storage: Everything is saved to your browser. No server is used.
